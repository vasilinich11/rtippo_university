## Прецендент "Сделать ставку на символ"

| | Сделать ставку на символ |
|---|---|
| Название | Ставка на символ |
| Описание | Игрок делает ставку на один из символов, представленных на колесе. |
| Основной исполнитель | Игрок |
| Предусловия | У игрока должно быть достаточно денег для совершения минимальной ставки. |
| Постусловия | Ставка игрока должна быть в пределах установленных рамок для минимальной и максимальной ставок. |


**Основной сценарий:**

| Действия актора | Реакция системы |
|---|---|
| Игрок увеличивает или уменьшает ставку. | Система принимает новое значение ставки от игрока. |
| | Система увеличивает (уменьшает) ставку в соответствии с действиями игрока. |
| | Ставка зафиксирована. |

**Альтернативы:**

| Действия актора | Реакция системы |
|---|---|
| Игрок увеличивает ставку | Система оставляет ставку неизменной, тк у Игрока недостаточно денег. |
| | Выводится сообщение об ошибке о недостатке средств на счете игрока. |

| Действия актора | Реакция системы |
|---|---|
| Игрок уменьшает или увеличивает ставку | Система оставляет ставку неизменной, тк превышены лимиты ставок. |
| | Выводится сообщение об ошибке о превышении лимитов ставок. |

| Действия актора | Реакция системы |
|---|---|
| Система начинает вращение колеса. | Колесо вращается до тех пор, пока не остановится на одном из секторов. |
|  | Система объявляет выигрышный сектор. |


## Прецендент "Получить выигрыш"

| | Выплатить выигрыш |
|---|---|
| Название | Выплата выигрыша |
| Описание | Система выплачивает выигрыш игроку в соответствии с результатами игры. |
| Основной исполнитель | Система |
| Предусловия | Определен выигрышный сектор. |
| | Игрок сделал ставки. |
| Постусловия | Выигрыш игрока выплачен. |

**Основной сценарий:**

| Действия актора | Реакция системы |
|---|---|
| Узнать выигрыш | Система определяет выигрышных игроков на основе выигрышного сектора и их ставок. |
| | Для каждого выигравшего игрока система вычисляет выигрыш в соответствии с таблицей выплат. |
| | Выигрыши выплачиваются каждому выигравшему игроку. |
| | Система добавляет выигрыши к счетам соответствующих игроков. |

**Альтернативы:**

| Действия актора | Реакция системы |
|---|---|
| Узнать выигрыш | Вывести выигрыш игрока на экран |
| Игрок не выигрывает. | Ничего не выплачивается. | 
| | Ставка игрока остается у казино. |

| Действия актора | Реакция системы |
|---|---|
| Узнать выигрыш | Вывести выигрыш игрока на экран |
| Определен выигрышный сектор, но у игрока недостаточно средств на счету для выплаты выигрыша. | Система выводит сообщение об ошибке о недостаточности средств на счете игрока. | 
| | Выигрыш остается у казино. |

## Прецендент "Баланс равен нулю"

| | Объявить банкротство игрока |
|---|---|
| Название | Баланс равен нулю |
| Описание | Система объявляет игрока банкротом, если он не может покрыть свои долги. |
| Основной исполнитель | Система |
| Предусловия | Игрок не может покрыть свои долги. |
| Постусловия | Игрок объявлен банкротом. |

**Основной сценарий:**

| Действия актора | Реакция системы |
|---|---|
| Проигрывает все деньги | Система определяет, что у игрока недостаточно средств для покрытия его долгов. |
| | Для каждого выигравшего игрока система вычисляет выигрыш в соответствии с таблицей выплат. |
| Игрок объявляется банкротом. | Все текущие ставки игрока утрачиваются в пользу казино. |

**Альтернативы:**

| Действия актора | Реакция системы |
|---|---|
| Игрок может покрыть свои долги. | Система не объявляет его банкротом. |
| Игрок не выигрывает. | Ничего не выплачивается. | 
| | Игрок продолжает игру. |
